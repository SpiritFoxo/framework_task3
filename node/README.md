# Практическое занятие №3 — Настройки запуска и базовая защита веб службы

## Быстрый старт

### Локальный запуск (Node.js ≥ 18)

```bash
npm install
npm start
```

Сервер запустится на порту **3000** в учебном режиме согласно `config/appsettings.json`.

---

## Режимы работы

Режим задаётся только настройкой — переписывать код не нужно.

### Учебный режим

Подробные сообщения об ошибках, мягкие лимиты по умолчанию.

**Через файл** (`config/appsettings.json`):
```json
{ "app": { "mode": "учебный" } }
```

**Через переменную окружения:**
```bash
APP_MODE=учебный node src/server.js
```

**Через аргумент командной строки** (наивысший приоритет):
```bash
node src/server.js --mode=учебный
```

---

### Боевой режим

Нейтральные сообщения об ошибках (`"Ошибка обработки запроса"`), строгие лимиты.

**Через переменную окружения:**
```bash
APP_MODE=боевой node src/server.js
```

**Через аргумент:**
```bash
node src/server.js --mode=боевой
```

---

## Порядок приоритета настроек

```
аргументы командной строки  >  переменные окружения  >  файл appsettings.json
```

Каждый следующий источник перезаписывает предыдущий.

---

## Все переменные окружения

| Переменная             | Описание                              | Пример                          |
|------------------------|---------------------------------------|---------------------------------|
| `APP_MODE`             | Режим работы                          | `учебный` / `боевой`            |
| `APP_PORT`             | Порт сервера                          | `8080`                          |
| `APP_TRUSTED_ORIGINS`  | Доверенные источники через запятую    | `http://localhost:5173`         |
| `APP_READ_PER_MINUTE`  | Лимит GET-запросов с одного IP        | `60`                            |
| `APP_WRITE_PER_MINUTE` | Лимит POST-запросов с одного IP       | `20`                            |

---

## Аргументы командной строки

Те же параметры, но с префиксом `--` и `=`:

```bash
node src/server.js --mode=боевой --port=8080 --trustedOrigins=https://mysite.ru --readPerMinute=30 --writePerMinute=5
```

## Docker

### Учебный режим

```bash
docker build -t pr3-app .
docker run -p 3000:3000 pr3-app
```

### Боевой режим

```bash
docker run -p 3000:3000 -e APP_MODE=боевой pr3-app
```

### Боевой режим с полными настройками

```bash
docker run -p 8080:8080 \
  -e APP_MODE=боевой \
  -e APP_PORT=8080 \
  -e APP_TRUSTED_ORIGINS=https://mysite.ru,https://api.mysite.ru \
  -e APP_READ_PER_MINUTE=30 \
  -e APP_WRITE_PER_MINUTE=5 \
  pr3-app
```

## Маршруты API

| Метод  | Путь                    | Описание                       |
|--------|-------------------------|--------------------------------|
| GET    | `/api/items`            | Список всех элементов          |
| GET    | `/api/items/by-id/:id`  | Получить элемент по ID         |
| POST   | `/api/items`            | Создать элемент                |
| GET    | `/api/mode`             | Текущий режим работы           |

**Создание элемента:**
```bash
curl -X POST http://localhost:3000/api/items \
  -H "Content-Type: application/json" \
  -d '{"name": "Тестовый товар", "price": 99.9}'
```
